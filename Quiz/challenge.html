Fragen über Web, HTML, CSS und Javascript.

Versuche die Fragen so gut es geht für dich zu beantworten, bevor du die Antworten anklickst.

Einsteiger Level:

1. Was ist JavaScript?
[Klick für Antwort] JavaScript ist eine leichtgewichtige, interpretierte Programmiersprache mit Objekt-Orientierten Fähigkeiten(OOP), die es uns erlaubt Interaktivität auf ansonsten statischen Webseiten zu ermöglichen. Vorraussetzung um JS zu nutzen ist lediglich ein Browser.

2. Was ist der Unterschied zwischen Java und JavaScript?
[Klick für Antwort] Java Code muss kompiliert werden. Daraus kann dann z.B. ein ausführbares Programm werden. JavaScript Code wird interpretiert. Zum Beispiel im Browser. Heutzutage kann JS auch auf einem System interpretiert werden, z.B. mit NodeJS. Die Ähnlichkeit der Namen der Sprachen haben keinerlei technischen Gründe.

3. Welche Datentypen existieren in JavaScript? Und was ist "Type-Casting"?
[Klick für Antwort]

Es existieren folgende Datentypen:

Undefined, Null, Boolean, String, Symbol, Number, Object

Unter "Type-Casting" versteht man das umwandeln von einem Typen in einen Anderen.


4. Was versteht man unter Responsive Webdesign?
[Klick für Antwort]

Derzeitiger Stand im Jahr 2022 ist, dass mehr Personen eine Webseite über mobile Geräte aufrufen, als über einem Desktop-Computer. Daher muss dafür gesorgt werden, dass die Seite ebenso auf kleinen, wie auf großen Screens gut dargestellt ist. Hierfür werden Breakpoints(z.B. Media Queries in CSS) verwendet.


5. Ist JavaScript "Case-Sensitive"? Was bedeutet das?
[Klick für Antwort]

Ja, JavaScript ist eine case sensitive Sprache. Die Keywords, Variablen, Funktionen-Namen, und andere Identifiers müssen immer konsistent in ihren kleinen oder großen Buchstaben geschrieben werden. Beispiel: "TEXT" ist eine andere Variable als "Text" oder "text" oder "tExT".


6. Wie erstellen Sie ein Objekt in JavaScript?
[Klick für Antwort]

Sie können in JavaScript ein Objekt mit der "object literal" wie folgt erstellen:

let person = { name: "Anton", age: 33 };

7. Wie erstellen Sie ein Array in JavaScript?
[Klick für Antwort]

Sie können in JavaScript ein Array mit der "array literal" wie folgt erstellen:

let reihe = [1, 2, 3, 4];

8. Was ist der Scope von Variablen in JavaScript?
[Klick für Antwort]

Der Scope einer Variablen ist die "Region" des Programmes, in welcher sie definiert ist, und somit verwendbar ist. JavaScript kennt zwei Arten von Scopes:

• Globale Variable − Eine globale Variable ist überall im Code verfügbar.

• Lokale Variable − Eine lokale Variable ist nur in der Funktion sichtbar in der sie definiert ist. Funktions-Parameter sind immer lokale Variablen.


9. Welche Bedeutung hat der "This" Operator in JavaScript?
[Klick für Antwort]

Das Keyword "This" in JavaScript bezieht sich auf das Objekt zu welchem es gehört. "This" hat verschiedene Werte, abhängig davon, wo es verwendet wird. In einer Methode(= Funktion eines Objektes) bezieht sich "This" auf das Besitzende Objekt und in einer Funktion auf das globale Objekt.


10. Was ist ein "Callback"?
[Klick für Antwort]

Ein Callback ist eine gewöhnliche JavaScript-Funktion, die an eine Methode als Argument oder Option übergeben wurde. Es ist eine Funktion, welche ausgeführt wird, wenn eine Andere Funktion ihre Ausführung beendet hat, daher der Name Call Back, Rückruf. Es klingt zunächst seltsam, aber Funktionen sind auch Variablen. Alles in JavaScript sind Objekte.


11. Was ist der "Closure"? Geben Sie ein Beispiel.
[Klick für Antwort]

In JavaScript können, anders als bei manchen anderen Sprachen, Variablen ausserhalb einer Funktion definiert werden, und dann in der Funktion verwendet werden. Man kann immer auf Variablen, die ausserhalb des Scope in dem man arbeitet liegen, zugreifen.

Also: Closures werden erstellt wann immer eine Variable, die Ausserhalb erstellt wird, von einem inneren Scope verwendet wird. Um einen Closure zu verwendet, erstelle eine Funktion in einer Funktion. Die innere wird auf das Äussere zugreifen können.
